# Improvision - Improvisation + Vision

## Description

Improvision is an interactive learning platform for practicing and understanding musical scales. The application helps musicians of all instruments to explore and internalize scales visually.

## Features

✨ **Comprehensive Scale Selection**
- Choose from various scales (Major, Minor, Pentatonic, Modes, and much more)
- All chromatic root notes available
- Flexible combination of scale and root note

🎲 **Random Generator**
- Random selection of scale and root note
- Perfect for training and testing your skills

🎹 **Multiple Instrument Visualizations**
- **Keyboard**: Classic keyboard representation
- **Guitar**: Guitar fretboard with fret markers
- **Bass**: Bass fretboard with 4 strings
- **Text Display**: All scale tones as letters

🎨 **Color Highlighting**
- Each scale tone is uniquely color-coded
- Easy identification of scale tones on all instruments

## Installation

1. Open the `index.html` file in your web browser
2. No additional installations or dependencies required

## Usage

1. **Select Scale**: Choose a scale from the dropdown menu
2. **Choose Root Note**: Select a root note from the list
3. **Display**: Click "Show Scale" to visualize the scale
4. **Switch Instrument**: Use the "Go to..." button to switch between Keyboard, Guitar, and Bass
5. **Random Mode**: Click "Random Scale" to generate a random scale and root note

## Technologies

- **HTML5**: Structure and SVG graphics
- **CSS3**: Styling and responsive design
- **JavaScript (Vanilla)**: Logic and interactivity

## Files

- `index.html` - Main file with HTML structure and SVG elements
- `script.js` - JavaScript logic with `Scale` and `ScaleView` classes
- `style.css` - Styling and layout
